# Pocket DM v3.1 — Build Guide

## Build APK with GitHub Actions (no install needed!)

### Step 1: Create GitHub repo
- Go to github.com → New repository → Name: PocketDM → Public → Create

### Step 2: Upload files
- Click "uploading an existing file"
- Drag ALL contents of this folder (app/, .github/, gradle/, build.gradle, settings.gradle, etc.)
- Commit changes

### Step 3: Get APK
- Click Actions tab → Wait for green checkmark (3-5 min)
- Click the build → Download "PocketDM-APK" from Artifacts
- Extract ZIP → PocketDM-v3.1.apk is your app!

### Step 4: Install
- Send APK to phone → Open → Install → Done!
